# 馬場天気 仕様書

## 1. 目的

乗馬競技会の現地天気を確認し、主に以下の判断材料として使う。

- 夜間の馬着をどうするか
- 翌日の試合に出すかどうか
- 現地入り後の気温、雨、平均風速の変化確認

判断メモ、馬着目安、競技場比較機能は現時点では対象外とする。

## 2. 想定ユーザー

- 乗馬クラブ関係者
- 選手、グルーム、輸送担当者
- 競技会期間中に現地気象をスマートフォンで確認したい人

## 3. 対象競技場

初期登録競技場は以下とする。

- つま恋
- 馬事公苑
- 御殿場
- 三木ホースランドパーク
- 山梨県馬術競技場
- ノーザンホースパーク

競技場は `src/data/venues.json` で追加、修正できる。
緯度経度は運用前に厩舎、競技場中心、馬場付近など実際に使いたい地点へ調整する。

## 4. 表示する気象データ

Open-Meteo Forecast API から時間別データを取得する。

| 項目 | 単位 | 用途 |
|---|---:|---|
| 気温 | ℃ | 夜間の冷え込み、日中の暑さ確認 |
| 体感温度 | ℃ | 風を含めた寒さ・暑さ確認 |
| 降水量 | mm/h | 雨の強さ確認 |
| 降水確率 | % | 雨の可能性確認 |
| 平均風速 | m/s | 風の強さ確認 |
| 相対湿度 | % | 補助情報 |

突風は表示しない。

## 5. 画面構成

### 5.1 ヘッダー

- アプリ名：馬場天気
- サブタイトル：乗馬競技場 時間別天気予報

### 5.2 操作エリア

- 競技場選択
- 表示時間枠選択
- 更新ボタン

### 5.3 状態表示

- 選択中の競技場
- 表示期間
- 現在時刻
- データ取得時刻

### 5.4 グラフ表示

グラフは3分割とする。

1. 気温・体感温度
2. 降水量・降水確率
3. 平均風速

グラフ上の任意の時間を触る、またはマウスオーバーすると、その時間の数値をツールチップで表示する。

### 5.5 毎時データ表

ページ下部に毎時の数値を表形式で表示する。
スマートフォンでは横スクロール可能にする。

表示項目は以下。

- 時刻
- 気温
- 体感温度
- 降水量
- 降水確率
- 平均風速
- 湿度

## 6. 表示時間枠

以下を選択可能とする。

| 表示モード | 内容 |
|---|---|
| 今日＋明日 | 今日0:00から明日23:00まで |
| 夜間〜翌日 | 今日18:00から明日18:00まで |
| 明日だけ | 明日0:00から明日23:00まで |
| 試合期間 | 開始日・終了日を日付指定 |
| カスタム | 開始日時・終了日時を指定 |

試合期間は3〜5日程度の競技会を想定する。

## 7. 現在時刻表示

現在時刻が表示範囲内にある場合、以下の表示を行う。

- グラフ：赤い縦線
- 毎時表：該当する時間の行を強調し、NOWバッジを表示

現在時刻は1時間単位に丸め、例として `14:23` の場合は `14:00` の行を現在行として扱う。

## 8. 技術仕様

| 項目 | 内容 |
|---|---|
| フロントエンド | React + TypeScript |
| ビルドツール | Vite |
| グラフ | Recharts |
| データ取得 | Open-Meteo Forecast API |
| 公開方法 | GitHub Pages + GitHub Actions |

## 9. データ取得仕様

取得先：`https://api.open-meteo.com/v1/forecast`

主なパラメータ：

```text
latitude
longitude
hourly=temperature_2m,apparent_temperature,precipitation,precipitation_probability,wind_speed_10m,relative_humidity_2m
timezone=Asia/Tokyo
wind_speed_unit=ms
precipitation_unit=mm
start_date=YYYY-MM-DD
end_date=YYYY-MM-DD
```

## 10. GitHubから派生させる運用

GitHubリポジトリを基点として、以下の派生運用を想定する。

- 競技場追加は `venues.json` を編集
- 表示項目追加は `openMeteo.ts` と `WeatherCharts.tsx` を編集
- デザイン変更は `styles.css` を編集
- GitHub Pagesで公開
- 将来的にPWA化、スマホホーム画面追加、競技場別URLなどを追加可能

## 11. MVP範囲外

現時点では以下は実装しない。

- 馬着判断メモ
- 気温別の判断目安
- 競技場比較
- 数ヶ月前のエントリー判断用の長期予報
- 突風表示
- ユーザー認証
- データ保存
